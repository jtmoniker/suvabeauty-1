Product Attribute Image

In this application new attribute type is added as 'Image'. when user select the attribute type as the image system is allow user to add the image in attribute value. Image is appears on ecommerce website.

It helpful to portal user (customer) to identify the product attribute (style). Also helpful to companies to grow his business.

Features

  • Add image in attribute value and show on website.
  • Customer can easily select attibute by selecting image on website.

Configuration

1) Select Attribute type as 'Image'.

2) Add Image in Attribute Values.

3) Add Attribute and Attribute values on product template.

In product page on website, customer can see attribute values as images and select any option.
Customer can see selected option on Cart page, Payment page and on order confirmation page.
